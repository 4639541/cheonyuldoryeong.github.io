# 천율도령 공식 신점 상담 홈페이지 3.5 안정화 버전

CSS 깨짐을 방지하기 위해 레이아웃을 다시 안정화한 버전입니다.

핵심:
- 모바일/PC 반응형 레이아웃 재정리
- 버튼 크기 자동 조절
- 화면 밖으로 버튼이 길게 늘어나는 문제 방지
- 프로필 카드 정상 표시
- 메뉴 버튼 정상 표시
- 기존 Firebase 기능 유지
- 관리자 모바일 관리 유지

GitHub에 압축 해제 후 전체 파일을 덮어쓰기 업로드하세요.
특히 index.html, style.css, app.js, admin.html, admin.js를 반드시 모두 바꿔야 합니다.

# 천명신당 공식 홈페이지 2.0

업로드 파일:
- index.html
- admin.html
- style.css
- firebase-config.js
- app.js
- admin.js

기능:
- 메인 홈페이지
- 상품 검색/카테고리/장바구니/바로 구매
- 주문 신청 및 관리자 주문 관리
- 상담 예약 및 관리자 예약 관리
- 상품 이미지 여러 장 업로드
- 게시판 글 + 이미지 여러 장 업로드
- 후기 사진 첨부 + 관리자 승인
- 공지사항
- 상담 분야 관리
- 관리자 대시보드 통계

필수 Firebase:
- Authentication 이메일/비밀번호
- Firestore Database
- Storage

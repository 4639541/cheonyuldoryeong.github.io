# 천율도령 공식 신점 상담 3.0 클린버전

완전히 새로 정리한 안정형 버전입니다.

포함 기능:
- 100% 반응형 메인 홈페이지
- 프로필 표시(profile.png)
- 카카오톡 문의 링크
- 상품 등록/구매/장바구니
- 주문번호 자동 생성
- 카카오페이 + 계좌번호 등록
- 쿠폰 등록/적용
- 상담 가격 등록/수정/삭제
- 상담 예약 달력
- 예약 가능 시간 관리자 설정
- 후기 사진 여러 장 업로드
- 관리자 승인/승인취소
- 게시판 글 + 이미지 업로드
- 배송조회 입력
- 주문/예약 상태 변경
- 모바일 관리자 탭 메뉴
- 문자/카카오톡 안내문구 복사

업로드 방법:
GitHub 저장소에 이 ZIP을 압축 해제하여 전체 파일을 덮어쓰기 업로드하세요.


# 3.1 전체 기능 반영

추가/정리 완료:
- 관리자에서 수정하면 Firebase 기준으로 홈페이지에 반영
- 모바일/PC 동일 반응형 화면
- 버튼 크기 자동 조절
- 이미지 자동 최적화 후 업로드
- 카카오페이/계좌 결제 정보 등록
- 주문번호 자동 생성
- 관리자 대시보드
- 예약 달력
- 배송조회 입력
- 쿠폰 등록/적용
- 재고 자동 차감
- 관리자 모바일 관리


수정 내용:
- 홈페이지 상단의 '홈페이지 2.0' 문구 제거
- 브랜드명은 '천율도령 공식 신점 상담'으로 통일

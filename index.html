공지사항

구매

구매는 카카오톡 문의 후 진행됩니다.

상품명, 이름, 연락처, 주소, 구매 개수를 보내주시면 결제 및 배송 안내를 드립니다.

배송

상품 확인 후 순차 발송됩니다.

상품 특성상 주문 확인, 포장, 발송 일정은 문의 시 안내드립니다.

안내

상징적 의미를 담은 상품입니다.

특정 효능이나 결과를 보장하지 않으며, 전통 문화와 상징적 의미를 담은 상품입니다.

SHOP

상품 구매

상품은 products.json 파일을 수정하면 직접 추가할 수 있습니다. 구매하기 버튼은 주문 양식을 보여주고 카카오톡으로 연결합니다.

ADMIN GUIDE

상품 직접 올리는 법

1. GitHub 저장소에서 products 폴더에 상품 이미지를 업로드합니다.

2. products.json 파일을 열고 상품명, 가격, 설명, 이미지 경로를 추가합니다.

3. 저장하면 홈페이지 상품 목록에 자동 반영됩니다.

예시 이미지 경로: products/product7.png
